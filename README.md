## 🔥 Key Features

### 🖼️ Image Toolkit
*   **High-Fidelity Codec Tuning**: Granular control over **AVIF**, **WEBP**, and **JPEG**. Toggle lossless modes, adjust chroma subsampling, and set encoding effort sliders to maximize quality[cite: 22].
*   **Smart Target Compression**: Built-in binary search algorithms to hit exact megabyte (MB) targets for images and PDF-wrappers without unnecessary quality loss[cite: 22].
*   **Hardware-Accelerated OpenCV**: Utilizes **OpenCL (GPU)** via your graphics card to handle heavy "Auto-Enhance" and deskewing tasks instantly[cite: 22].
*   **Elite Watermarking**: Bulletproof diagonal tiling stamps your watermark in a rotated grid across the entire image, preventing simple cropping[cite: 22].
*   **EXE Icon Generation**: Quickly convert images to multi-size bundled `.ico` files for Windows applications[cite: 22].

### 📄 Document Toolkit (PDF & DjVu)
*   **Interactive Visual Rearranger**: Before merging, a high-resolution grid pops up with live thumbnails[cite: 26]. Drag-and-drop to reorder chapters or images visually before committing the build[cite: 26].
*   **Searchable PDF (OCR)**: Integrates the **RapidOCR** engine to inject invisible text layers into image-based PDFs, making your documents fully searchable[cite: 22].
*   **Non-Destructive Compression**: Native PDF stream deflation and garbage collection that shrinks files without flattening them into low-quality images[cite: 22].
*   **DjVu Integration**: Full support for DjVu to PDF (and vice versa) conversions (requires `djvulibre`)[cite: 22].
*   **Advanced Security**: AES-256 password protection and customizable author metadata injection[cite: 22].

### ⚡ Power-User Experience
*   **Persistent Workflows**: The app features a "brain" that auto-saves and restores your last-used settings, sliders, and even your last browsed directory[cite: 22, 23, 27].
*   **Zero-Config Startup**: An integrated auto-bootstrapper checks for and installs missing dependencies via `pip` the moment you run the script[cite: 22].
*   **Animated UI**: Features a synchronized "breathing" rainbow header and progress bar for that signature "Alvin Rocks" look[cite: 14, 18].
*   **Drag & Drop**: Native PySide6 drag-and-drop support for effortless batch loading[cite: 14].
